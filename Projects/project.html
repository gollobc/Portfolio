Projects

WordWise

A MERN stack app styled with Semantic-UI. Multiple 3rd party API connections that allow the user to search words and some phrases and display information such as definitions and synonyms. User can save words to collections for later reference.

Meridio

Django app designed to be a community marketplace. Users have the ability browse a gallery of all items posted to marketplace. Foreign keys are used to associate bids with items and contact forms with bids. Simple, stylish, and convenient interface achieved with CSS.

ActiveInsights

A MEN stack app with full CRUD capability. Fitness focused and community centered with a comment board. Users will find basic information relating to health and fitness, along with links to extra resources.
